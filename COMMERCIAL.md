# 상업적 사용 안내

Kytos OS는 AGPL v3 라이센스로 배포됩니다.

---

## 무료 사용 (AGPL v3)

다음의 경우 Kytos OS를 무료로 사용할 수 있습니다.

- 개인이 자신의 조직에 사용하는 경우
- 비영리 단체가 사용하는 경우
- 학술 연구 목적으로 사용하는 경우
- 수정본을 AGPL v3으로 공개하는 경우
- 포크하여 오픈소스로 배포하는 경우

---

## 상업적 라이센스 필요

다음의 경우 별도의 상업적 라이센스가 필요합니다.

- Kytos OS를 기반으로 유료 서비스를 제공하는 경우
- Kytos OS를 포크하여 소스를 비공개로 유지하는 경우
- Kytos OS를 기반으로 제품을 판매하는 경우
- Kytos OS를 기반으로 컨설팅을 제공하고 보수를 받는 경우

---

## 상업적 라이센스 신청

hello@spiratus.org 로 문의해주세요.

조직 규모와 사용 목적을 알려주시면 적합한 조건을 협의합니다.

---

## Mycelium Network 참여자 혜택

Kytos OS로 만들어진 조직이 Mycelium Network에 참여하면
상업적 라이센스 조건이 달라질 수 있습니다.

Mycelium Network는 Kytos OS를 사용하는 조직들이
서로 연결되어 정보와 자원을 나누는 망입니다.
생태계에 기여하는 만큼 혜택이 돌아오는 구조를 지향합니다.

---

*상업적 라이센스 수익은 Kytos OS 프로젝트의 지속적인 개발에 사용됩니다.*
